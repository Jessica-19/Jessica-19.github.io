如何使用markdown语法

在vscode 里面用markdown

1-下载markdown all in one 和 Markdown Preview Github Styling

2-新建一个文件，用vscode 打开 新建一个文件尾缀 .md 可以看到文件前面的标志改变说明成功创建

标题和正文

代表一级标题#，此类推，二级三级标题

正文前什么都不加

或者在文字下方输入===为一级语法 —为二级语法（有点限制）

代码语块

在这里```后面可以接上代码语言类型，两个来表示开头和结尾

有序列表，输入数字，加一个句点，然后空格即可；在顺序的数字前可以tab缩进空置出现多级列表

  1. 你好
  2. 我好
    1. 大家好
    2. 我们好
  3. 太好了

无序列表，输入 - ,然后空格

  • 你好

基于vscode编写markdown导出带书签(目录)的pdf

  1. 下载prince

https://www.princexml.com/download/

有点没安装好，vscode无法直接响应，需要enginer/bin/文件的path，修改系统环境变量新增path，再重启vscode。但是我这版只安装了exe文件和证书，只能先导出html再用prince转换到pdf，效果一样的。

  1. vscode 安装插件 markdown preview enhanced

这个插件才可以搭配导出pdf格式，markdown github stylying 不太行 两者显示也是有区别的

鼠标右键 预览模式下选择
3. 导出markdown 的html格式或者pdf 文件
4. 在最前面 插入目录
光标放在在正文最前面,然后输入shift+ctrl+p,搜索 toc选择
选项(markdown preview enhanced: create toc),然后 ctrl+s保存文件
